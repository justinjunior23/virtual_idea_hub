# Virtual Idea Hub — Multi-Role Submission Management System

## Project Overview

Virtual Idea Hub is a web-based multi-role submission management platform developed to streamline the collection, review, and resolution of reports, ideas, and organizational submissions.

The system enables three main user roles to interact through a structured workflow:

* **Users** submit reports or ideas through categorized forms.
* **Staff members** review submissions and provide solutions or responses.
* **Administrators** oversee the entire system, monitor activity, and manage users.

This project demonstrates full-stack web development skills using Django, including role-based authentication, dashboard design, workflow management, and database-driven operations.

## System Workflow

1. A registered user logs into the platform.
2. The user submits a report under a selected category.
3. Staff members access submitted reports.
4. Staff review the issue and provide a solution.
5. Admin monitors all activities and performance statistics.
